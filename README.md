# Probiotika — Tienda V5

Primera reconstrucción limpia del sitio de Probiotika, basada en el boceto aprobado.

## Incluye
- Homepage responsive.
- Hero orientado a conversión.
- Scobyto como guía, sin bloquear la compra.
- Productos destacados.
- Carrito de demostración con `localStorage`.
- Pack descubrimiento.
- Beneficios, historia, categorías, mayoreo y FAQ.
- Espacios preparados para fotografías reales.
- Estructura compatible con una futura migración a Shopify.

## Probar en GitHub Pages
1. Sube todo el contenido de esta carpeta a la raíz del repositorio.
2. En GitHub abre **Settings → Pages**.
3. Selecciona **Deploy from a branch**.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda.

La página debería quedar disponible en:
`https://hola702.github.io/Probiotika/`

## Archivos
- `index.html`: estructura de la homepage.
- `assets/css/styles.css`: diseño y responsive.
- `assets/js/app.js`: catálogo, carrito y microinteracciones.
- `assets/images/`: colocar aquí fotos optimizadas.

## Pendiente antes de publicar comercialmente
- Sustituir botellas ilustradas por fotografías reales.
- Confirmar precios, cobertura, envío gratis y políticas.
- Añadir reseñas reales.
- Configurar dominio.
- Convertir el prototipo en tema de Shopify.
- Conectar Shopify Checkout.
- Instalar GA4, Meta Pixel y eventos de comercio electrónico.
- Crear fichas individuales de producto.
